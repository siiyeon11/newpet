# 메이플스토리 캐릭터 검색 & 랭킹 웹페이지

넥슨 Open API를 사용하여 메이플스토리 캐릭터 정보를 조회하고 월드 랭킹을 표시하는 웹 애플리케이션입니다.

## 🚀 시작하기

### 1. 의존성 설치

```bash
npm install
```

### 2. 프록시 서버 실행

```bash
npm start
```

서버가 `http://localhost:3000`에서 실행됩니다.

### 3. 브라우저에서 접속

브라우저에서 `http://localhost:3000`을 열어주세요.

## 📝 사용 방법

1. **API Key 입력**
   - 넥슨 Open API Key를 입력하고 "저장" 버튼을 클릭합니다.
   - API Key는 자동으로 저장되어 다음에 다시 입력할 필요가 없습니다.

2. **캐릭터 검색**
   - 캐릭터 이름을 입력하고 "검색" 버튼을 클릭합니다.
   - 캐릭터의 기본 정보(레벨, 직업, 길드명 등)가 표시됩니다.

3. **월드 랭킹 조회**
   - 월드를 선택하고 "랭킹 조회" 버튼을 클릭합니다.
   - 선택한 월드의 상위 100위 랭킹이 표시됩니다.

## 🛠️ 기술 스택

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **API**: 넥슨 Open API

## ⚠️ 주의사항

- 넥슨 Open API는 브라우저에서 직접 호출할 경우 CORS 오류가 발생할 수 있습니다.
- 이 프로젝트는 프록시 서버를 통해 CORS 문제를 해결합니다.
- 반드시 프록시 서버(`npm start`)를 실행한 후 사용해주세요.

## 🔧 문제 해결

### 캐릭터 검색이 실패하는 경우

1. API Key가 올바르게 입력되고 저장되었는지 확인하세요.
2. 프록시 서버가 실행 중인지 확인하세요 (`npm start`).
3. 브라우저 개발자 도구(F12)의 콘솔에서 오류 메시지를 확인하세요.
4. 넥슨 Open API Key가 유효한지 확인하세요.

### CORS 오류가 발생하는 경우

- 프록시 서버를 통해 API를 호출하도록 설정되어 있습니다.
- `server.js` 파일이 실행 중인지 확인하세요.
- 브라우저에서 `http://localhost:3000`을 통해 접속해야 합니다.
