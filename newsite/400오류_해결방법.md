# 🔧 400 오류 해결 방법

## 400 오류란?

HTTP 400 오류는 "Bad Request"를 의미하며, 서버가 요청을 이해할 수 없다는 뜻입니다.

## 📋 체크리스트

### 1. API Key 확인
- [ ] API Key가 입력되어 있나요?
- [ ] "저장" 버튼을 눌렀나요?
- [ ] "✓ API Key가 저장되었습니다" 메시지가 보이나요?
- [ ] API Key에 공백이나 불필요한 문자는 없나요?

### 2. 캐릭터 이름 확인
- [ ] 캐릭터 이름을 입력했나요?
- [ ] 캐릭터 이름에 특수문자는 없나요?
- [ ] 띄어쓰기나 오타는 없나요?

### 3. 서버 상태 확인
- [ ] 프록시 서버가 실행 중인가요?
- [ ] 브라우저 주소가 `http://localhost:8000`인가요?
- [ ] 브라우저를 새로고침했나요?

## 🔍 단계별 해결 방법

### 방법 1: API Key 다시 입력

1. **기존 API Key 삭제**
   - 브라우저 개발자 도구(F12) 열기
   - Console 탭에서 다음 명령 실행:
     ```javascript
     localStorage.removeItem('maplestory_api_key');
     location.reload();
     ```

2. **API Key 다시 입력**
   - 새로운 API Key 입력
   - "저장" 버튼 클릭
   - 저장 확인 메시지 확인

### 방법 2: 넥슨 API Key 확인

넥슨 개발자 센터에서 확인:
- https://open.nexon.com/ 접속
- 로그인 후 "내 애플리케이션" 메뉴 확인
- API Key가 활성화되어 있는지 확인
- API Key를 새로 발급받아보세요

### 방법 3: 브라우저 콘솔 확인

1. **브라우저 개발자 도구 열기**
   - F12 키 누르기

2. **Console 탭 확인**
   - 빨간색 오류 메시지 확인
   - "요청 URL" 로그 확인
   - "응답 상태" 로그 확인

3. **Network 탭 확인**
   - 캐릭터 검색 버튼 클릭
   - 요청이 실패한 항목 클릭
   - "Response" 탭에서 실제 오류 메시지 확인

### 방법 4: 서버 로그 확인

프록시 서버가 실행 중인 터미널 창에서:
- `[프록시 요청]` 로그 확인
- `[응답 에러]` 로그 확인
- 오류 메시지 확인

## 💡 일반적인 원인과 해결책

### 원인 1: API Key가 전달되지 않음
**증상:** 프록시 서버 로그에 "API Key가 전달되지 않았습니다" 메시지

**해결:**
1. API Key 입력 필드 확인
2. "저장" 버튼 다시 클릭
3. 브라우저 새로고침

### 원인 2: API Key가 유효하지 않음
**증상:** 넥슨 API에서 "Invalid API Key" 오류

**해결:**
1. 넥슨 개발자 센터에서 API Key 확인
2. 새로운 API Key 발급
3. 새 API Key로 다시 저장

### 원인 3: 캐릭터 이름 문제
**증상:** "캐릭터를 찾을 수 없습니다" 오류

**해결:**
1. 캐릭터 이름 정확히 입력
2. 대소문자 확인
3. 특수문자 제거

### 원인 4: 요청 형식 문제
**증상:** 프록시 서버에서 400 오류

**해결:**
1. 브라우저 새로고침 (F5)
2. 서버 재시작
3. 캐시 삭제 (Ctrl+Shift+Delete)

## 🆘 그래도 안 되면

다음 정보를 확인해주세요:

1. **브라우저 콘솔의 전체 오류 메시지**
   - F12 → Console 탭
   - 빨간색 오류 메시지 전체 복사

2. **프록시 서버 로그**
   - 서버 터미널의 오류 메시지

3. **사용 중인 브라우저**
   - Chrome, Edge, Firefox 등

4. **API Key (일부만)**
   - 앞 4자리와 뒤 4자리만 (예: `abcd...wxyz`)
